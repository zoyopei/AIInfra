# 生成式推荐

大模型时代一切阶生成，其中把传统的推荐也用大模型来生成。生成式推荐是一种利用生成模型来生成个性化推荐内容的技术。它通过分析用户的历史行为和偏好，学习用户兴趣的潜在表示。
