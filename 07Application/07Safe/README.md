# AI 安全

隐私计算是一种在保护数据隐私的前提下进行数据计算的技术。其核心在于实现“数据可用不可见”，即在不泄露原始数据的情况下，完成数据的分析和计算。一起看看大模型时代隐私计算会遇到什么冲击。
